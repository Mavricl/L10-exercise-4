# Hello this is my app

-----------

## What is this app about?
This app allows League of Legends players (and new learners) to easily browse champions by their roles such as Mage, Top, and Jungle.
Users can view champion arts, search by role, and even add their own champions using champion name + unique Splash Art code.

Perfect for fans who want a simple mobile LoL reference app.

### Notable features of the app
Browse champions by roles

Champions are grouped into Mage, Top, and Jungle categories.

Each section includes a title and FontAwesome icon.

Add New Champions

Users can add their own champions by entering:

Champion Name

Identifier Code (used to fetch the champion’s splash art)

Role (via a dropdown picker)

Live Champion Artwork Preview

Champion cards display splash arts dynamically using Riot’s CMS image links.

Modern Clean UI

Themed after League of Legends (dark mode + gold accents).

Smooth layout using React Native + Expo.
